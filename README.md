## 📌 Sobre o projeto

O **Wide Coverage Location** é uma landing page moderna desenvolvida para simular um serviço de mobilidade urbana.

O foco do projeto está na criação de uma interface limpa, responsiva e com boa experiência do usuário, destacando a proposta de valor de forma clara e objetiva.

## 💻 HTML - Estrutura

O projeto utiliza uma estrutura semântica e organizada:

- `<header>` → Navegação com links e botão de cadastro  
- `<main>` → Conteúdo principal  
- `<section>` → Área principal (hero section)  
- `<img>` → Ilustração  
- `<div>` → Agrupamento de conteúdo  
- `<h1>` → Título principal  
- `<p>` → Descrição  
- `<a>` → Call to Action

## 🎨 CSS - Estilização

A estilização foi feita com foco em design moderno e responsivo:

### ✨ Técnicas utilizadas:

- Reset CSS com `*`
- Uso de **Flexbox** para layout
- Tipografia com **Poppins**
- Espaçamentos bem definidos
- Hierarquia visual clara

### 🧠 Experiência do Usuário (UX)

- Implementação de estados interativos com `:hover` e `:active`
- Feedback visual imediato para ações do usuário
- Melhoria na usabilidade e percepção de interatividade

### 🎯 Destaques:

- Uso de `flex` com `justify-content` e `align-items`
- Aplicação de `clamp()` para fontes responsivas
- Layout adaptável com `flex-wrap`
- Botões e links com identidade visual consistente

---

## 📱 Responsividade

O projeto foi desenvolvido para diferentes tamanhos de tela:

### 🔹 Mobile (até 900px)

- Menu simplificado (links ocultos)
- Botão centralizado
- Layout em coluna
- Conteúdo centralizado
- Ajuste de fontes e espaçamento

---

### 🔹 Tablet / Médio (901px até 1420px)

- Ajuste de proporções da imagem
- Reorganização do layout
- Melhor uso do espaço horizontal
- Redimensionamento de textos

---

### 🔹 Desktop

- Layout em linha (imagem + conteúdo)
- Uso completo da largura da tela
- Melhor distribuição visual
